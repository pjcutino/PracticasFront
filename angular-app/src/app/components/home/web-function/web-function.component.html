This webpage displays all original 151 pokemon and lets you add or modify a Mega-evolution following this steps:

Step 1:

Create a Mega in the Forms view defining its name, image url and the base pokemon name to evolve it from.

Step 2:

Look up all Mega-evolutions on the list displayed in My Creates with the possibility to modify or delete them.

Step 3:

Search for the base pokemon on the list and the mega will be displayed on the up-right corner.
